Prioritask

An online web application that allows users to manage their tasks efficiently via a simple dashboard!
Visit Prioritask

Project Purpose

This web-based task management application was my project for my Senior Design at University of Maryland Eastern Shore (UMES). I led a team as project manager and technical lead in designing, implementing, testing, and deploying this application that allows a user to register an account with a username and password, login into their account, and add, delete, and edit tasks via a simple, yet effective dashboard, organize, filter, and sort these tasks, and even ask Taski, an AI-powered task assistant bot for helpful ways to solve a task! By creating this full-stack web application we were able to further practice working in teams, agile development, using HTML/CSS as a Front-End, Python as a Back-End, SQLite as a database management system, and integrating APIs and AI into a working application!

Future Plans

This web application has been a very educating and fun project to create, however, I would consider this project a protype and not ready to sustain actual real-world users, and serves more as a practical demonstration of what I can do in a team. I plan to take what I learned here, and use it as a base to learn more about web development, and create a more robust and feature-rich task management application in the future!
